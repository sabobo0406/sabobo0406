# 学習メモ（教訓ファイル）の書式

長時間タスクやセッションをまたぐ作業で得た教訓をここに蓄積する。
チャット履歴のコピーではなく、次のセッションで再利用できる形の教訓だけを書く。

## ルール

- **1教訓 = 1ファイル**。ファイル名は `YYYY-MM-DD-短いスラッグ.md`
- **先頭1行は教訓のサマリ**（太字）。一覧をざっと読むだけで内容が分かるようにする
- 本文には「状況 → 何が起きたか → 次からどうするか」を簡潔に書く
- 古くなった教訓は削除してよい（溜め込まない）

## テンプレート

```markdown
**サマリ: ここに教訓を1行で書く**

- 状況: 何をしていたか
- 起きたこと: 何がうまくいかなかった／分かったか
- 次から: どう行動を変えるか
```
