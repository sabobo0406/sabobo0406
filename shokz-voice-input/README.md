# Shokz イヤホンで AI にハンズフリー音声入力するセットアップ

「キーボードを押しながら話す」のではなく、**手元のボタン(または Shokz 本体のボタン)を押すと、Shokz のマイクから音声入力が始まる**仕組みを作るためのガイドです。

結論: **できます。** 方法は大きく3つあり、組み合わせも可能です。

---

## 方法A: 追加ハードなし — Shokz 本体のマルチファンクションボタンを使う(Mac)

Shokz の再生/一時停止ボタンは、Mac には「メディアキー」として届きます。
[Hammerspoon](https://www.hammerspoon.org/)(無料)でこのイベントを横取りして、「音楽の再生/停止」の代わりに「音声入力の開始/停止」に割り当てます。

- メリット: 何も買わなくていい。ボタンは常に耳元にあるので、部屋の中を歩き回りながら使える(Bluetooth が届く約10mの範囲)。
- デメリット: 「押している間だけ」ではなく「1回押すと開始、もう1回押すと終了」のトグル式になる(ヘッドセットのボタンは長押しが別機能に割り当てられているため)。

### セットアップ手順

1. Hammerspoon をインストール: `brew install --cask hammerspoon`
2. このフォルダの [`hammerspoon/init.lua`](hammerspoon/init.lua) を `~/.hammerspoon/init.lua` にコピー
3. Hammerspoon を起動し、アクセシビリティ権限を許可
4. 音声入力アプリ側のホットキーを `init.lua` の設定に合わせる(下記「音声入力アプリ」参照)
5. システム設定 → サウンド → 入力 で **Shokz をマイクに選択**

これで Shokz のボタンを押すと音声入力が始まり、もう一度押すと確定します。

### Typeless を使う場合の設定(おすすめ)

[Typeless](https://www.typeless.com/) は AI 整形つきの高精度な音声入力アプリです。方法A と組み合わせる手順:

1. Typeless をインストールして起動(初回のマイク・アクセシビリティ権限を許可)
2. Typeless の設定 → ホットキーで、**カスタムホットキーを `⌘⌥D`(Command + Option + D)** に変更
3. 録音モードを **「トグル(1回押すと開始 / もう1回で終了)」** にする
   - ※ Shokz のボタンは「1回押す」信号しか出せないので、Typeless 標準の「Push-to-Talk(押している間だけ)」モードは Shokz では使えません。トグルを選んでください。
   - ※「押している間だけ」を Shokz でやりたい場合は **方法B** を参照
4. `init.lua` は初期状態(`MODE = "hotkey"`、`HOTKEY = ⌘⌥D`)のままで OK
5. システム設定 → サウンド → 入力 で **Shokz をマイクに選択**

→ Shokz のボタンを押す → Typeless が録音開始 → 話す → もう一度ボタンを押す → 文字がカーソル位置(Claude / ChatGPT のチャット欄など)に入力されます。

## 方法B: 「押している間だけ話す」を実現 — 安い Bluetooth ボタンを使う(Mac)

完全に希望どおりの「ボタンを押している間だけ音声入力」(Push-to-Talk)をやるなら、Bluetooth ボタンを1個用意するのが確実です。

| デバイス | 価格目安 | 備考 |
|---|---|---|
| Bluetooth カメラシャッターリモコン (AB Shutter 3 など) | 500〜1,000円 | 「音量+キー」として認識される。Karabiner で別キーに変換 |
| 8BitDo Micro / Zero 2 | 約3,000円 | キーボードモードあり。複数キーを送れる |
| Flic 2 | 約4,000円 | 専用 Mac アプリで押す/長押しに直接アクションを割当可 |

### 仕組み

1. ボタンを Mac とペアリング(キーボードとして認識される)
2. [Karabiner-Elements](https://karabiner-elements.pqrs.org/)(無料)で、ボタンが送るキー(例: 音量+)を `F13` などに変換
3. 音声入力アプリ(Superwhisper / Wispr Flow / VoiceInk)の **ホールドモード(押している間だけ録音)** のホットキーに `F13` を設定
4. マイク入力を Shokz に設定

→ ポケットの中のボタンを**押している間だけ** Shokz のマイクで話せて、離すと文字が入力されます。Bluetooth の届く範囲(約10m)ならどこからでも OK。

## 方法C: iPhone の場合

iPhone は実はボタンすら不要にできます。

- **ChatGPT / Claude アプリの音声モード**: 一度開始すればハンズフリーで会話継続。Shokz のマイクがそのまま使われる
- **アクションボタン**(iPhone 15 Pro 以降): ショートカット経由で「ChatGPT 音声モードを開く」などを割当
- **背面タップ**(設定 → アクセシビリティ → タッチ → 背面タップ): iPhone の背面を2回叩くとショートカット起動
- キーボードの音声入力もマイクボタン1タップで開始でき、Shokz のマイクが使われる

## 音声入力アプリの選択肢(Mac)

文字起こしして AI のチャット欄に入力するためのアプリです。どれも「カーソルがある場所」にテキストが入るので、Claude や ChatGPT のウィンドウを開いておけばそこに入ります。

| アプリ | 価格 | ホールド(押す間だけ)対応 | 備考 |
|---|---|---|---|
| macOS 標準の音声入力 | 無料 | ✕(トグルのみ) | 設定 → キーボード → 音声入力 で有効化 |
| [VoiceInk](https://tryvoiceink.com/) | 無料(OSS) | ○ | ローカル Whisper で高精度。日本語OK |
| [Superwhisper](https://superwhisper.com/) | 無料枠あり | ○ | 人気。AIによる整形機能つき |
| [Wispr Flow](https://wisprflow.ai/) | 有料 | ○ | 精度・速度とも良い |

---

## 注意点

- **Shokz を Mac に接続しておくこと**: マルチポイント対応モデル(OpenComm, OpenRun Pro など)なら iPhone と Mac に同時接続できます。マルチファンクションボタンの信号は「最後に再生操作したデバイス」に飛ぶので、方法Aを使うときは Mac 側で使う想定にしてください。
- **マイク使用中は音質が落ちる**: Bluetooth の仕様(HFP プロファイル)上、マイクを使っている間は再生音質がモノラル・低音質になります。音声入力をやめると元に戻ります。
- **届く距離は Bluetooth の範囲(約10m、壁があると短くなる)**。家全体をカバーしたい場合は iPhone 側(方法C)と併用するのがおすすめ。
