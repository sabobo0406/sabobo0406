# 05. ミールプレップ・キングスの日曜バッチクッキング・システム

## 用途
日曜日の2〜3時間のセッションで1週間分の食事をすべて調理する効率的なシステムを設計する。

## プロンプト

```
あなたはミールプレップの効率化の専門家で、忙しいプロフェッショナルが日曜日の2〜3時間のセッションで1週間分の食事をすべて調理するのを手伝います — ほとんどの人はモチベーションが高い日曜日に栄養計画を失敗させるのではなく、疲れ果ててUber Eatsがワンタップ以内の水曜日の午後7時に失敗します。

私は、健康的な食事が1週間ずっと最も簡単な選択肢になる完全な日曜ミールプレップ・システムが必要です。

準備：

- バッチクッキングのタイムライン：日曜日の準備セッションのための分単位のスケジュールで、最大限の効率のために何をどの順番で調理するかを正確に示す
- 同時調理：オーブン、ストーブトップ、炊飯器に同時に何を入れるかで、決して待たずに済むようにする
- プロテインバッチ：2〜3種類のプロテインを大量に調理（5ポンドの鶏肉、2ポンドの七面鳥のひき肉、12個の卵）で、多様性のために異なる調味料を使用
- 炭水化物バッチ：2〜3種類の炭水化物源を準備（米、さつまいも、パスタ）で、量を計って再加熱する準備をする
- 野菜バッチ：2〜3種類の野菜をロースト、蒸す、または生で準備して1週間分にする
- ソースと調味料の多様性：3〜4種類の異なるソースや調味料で、同じベース食材が毎日完全に異なる味わいになるようにする
- 容器システム：必要な容器の数、サイズ、そしてグラブアンドゴーの利便性のためにラベル付けの方法
- 保管ガイド：冷蔵で5日持つもの、週の後半のために冷凍が必要なもの、そして柔らかくならないための適切な保管方法
- 再加熱の指示：各食事を新鮮に調理したように味わうための再加熱方法で、電子レンジで加熱したゴムのようなものではないようにする
- 器具リスト：バッチクッキングを2倍速くする正確なキッチンツール（天板、炊飯器、インスタントポット、食品スケール）

日曜ミールプレップのプレイブック形式で、時間指定の調理スケジュール、レシピリスト、容器計画、保管指示を含めてフォーマットしてください。

私の好み：[あなたのプロテインの好み、1日あたりの食事数、お持ちのキッチン器具、そして準備に費やせる時間の数を記述してください]
```

## 入力すべき情報

- 好きなタンパク質（鶏肉／牛肉／豚肉／魚／豆腐など）
- 1日あたりの食事数
- 所有しているキッチン器具（オーブン／インスタントポット／エアフライヤーなど）
- 準備に費やせる時間
- 家族構成（1人分／家族分）
