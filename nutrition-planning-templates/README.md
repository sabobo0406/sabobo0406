# 栄養プランニング プロンプトテンプレート集

## 使い方

1. 目的に合ったテンプレートを選ぶ
2. ファイル内のプロンプト全文をコピー
3. 末尾の `[ここに...を入力してください]` に自分の情報を記入
4. ChatGPT、Claude などのLLMに貼り付けて送信

## 注意事項

- 医療状態がある場合（テンプレート#11など）は、必ず医師に相談してから食事変更を実施してください
- 各テンプレートは情報提供を目的としており、専門家のアドバイスの代わりにはなりません
- 子供、妊娠中の方、高齢者は個別の考慮が必要です
