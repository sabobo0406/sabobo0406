# 03. ホールフーズ予算最適化食料品リスト作成ツール

## 用途
栄養を犠牲にせず、コストを最小限に抑えた食料品リストを生成する。

## プロンプト

```
あなたはシニア栄養経済学者で、家族が予算内で健康的に食事を摂るのを手伝います — 健康的に食べないための第1の言い訳は『健康食品は高すぎる』ですが、それは単に本当ではありません。戦略的に買い物する方法を知っていれば、です。

私の食事プラン用の完全な食料品リストが必要です。栄養を犠牲にせずにコストを最小限に抑えたもの。

最適化項目：

- 材料の統合：私の食事プランからすべての材料を1つのマスターショッピングリストにまとめ、店舗セクション別に整理
- 予算見積もり：週1回の食料品購入の推定総コスト（米国平均食料品価格に基づく）
- まとめ買い識別：頻繁に使用され、保存が効くアイテムを大量に購入すべきもの
- 店舗ブランド交換：栄養が同一で30-40%節約できる、特定のブランド商品を店舗ブランドに置き換え
- 今月の旬の産物ガイド：今月旬の果物や野菜で最も安いもの
- 冷凍 vs 新鮮分析：栄養的に同一の冷凍産物で50%安いもの（ほとんどの野菜、ほとんどのベリー類）
- タンパク質コスト最適化：1gあたりのタンパク質で最も安いタンパク源（鶏もも肉は胸肉より優れ、卵はすべてを上回る）
- 廃棄削減計画：購入したすべての材料を複数の食事で使い、冷蔵庫の奥で腐らないようにする方法
- パントリー常備品リスト：特別な買い物なしでどんな食事プランも可能にする、常に手元に置くべき20アイテム
- 月間節約計算：この最適化された買い物アプローチが、計画なしの食料品購入に比べてどれだけ節約できるか

店舗セクション別に整理された予算最適化食料品リスト形式で、数量、推定コスト、週総予算を含めてフォーマットしてください。

私の食事プラン：[ここに週の食事プランを貼り付け、または典型的な食事、世帯人数、週の食料品予算を記述]
```

## 入力すべき情報

- 週の食事プラン（またはテンプレート#02の出力）
- 世帯人数
- 週の食料品予算
- 買い物できる店舗（ホールフーズ／コストコ／地元スーパーなど）
- 保存スペース（冷凍庫サイズなど）
