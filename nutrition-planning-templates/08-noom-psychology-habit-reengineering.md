# 08. Noom 心理学ベースの食事習慣再構築

## 用途
「何を食べるか」ではなく「なぜ食べるか」を修正する。感情的食い、過食、夜食など行動パターンを再構築する。

## プロンプト

```
あなたはNoomのシニア行動心理学者で、不健康な食事の背後にある習慣を修正します — なぜなら、あなたの食事法が失敗するのは食べ物のせいではなく、行動パターン、感情的な引き金、そして環境的な合図が原因で、空腹でないときに食べたくなるからです。

私は過食の理由を修正する行動戦略が必要です。ただ何を食べるかではなく。

再構築：

- 引き金特定：過食を引き起こす具体的な状況、感情、一日の時間帯、環境的な合図（ストレス、退屈、社会的プレッシャー、視覚的な合図、習慣のループ）
- 感情的食事監査：空腹だから食べているのか、それともストレス、不安、孤独、退屈、またはお祝いのためか
- 環境再設計：キッチン、オフィス、日常環境に具体的な変更を加えて、健康的な選択をデフォルトにし、不健康的な選択に努力を要するようにする
- 習慣ループ分析：私の3つの最悪の食事習慣に対する合図 → ルーチン → 報酬のマッピングと、具体的な置き換えルーチン
- 量制御戦略：小さな皿の使用、事前分量分けのスナック、視覚的な盛り付けガイドを活用し、意志力に頼らない
- マインドフルイーティングプロトコル：食べ方を遅くし、満足感を高め、制限なしで摂取量を20%削減する具体的な実践
- 社会的食事戦略：レストラン、パーティー、家族の夕食を扱う方法で、過食したり疎外感を感じたりせずに
- 夕食後食欲修正：夕食後の間食を管理する具体的なプロトコル（ほとんどの人にとって余分なカロリーの第1の原因）
- 週末プラン：ほとんどの人は週末に300-500カロリー多く摂取する — 週の進捗を台無しにせずに週末を楽しむ戦略
- 責任システム：強迫的なカロリー計算なしで一貫性を保つ、2分間の毎日の追跡習慣

トリガーマッピング、習慣置き換え戦略、および30日間の習慣構築カレンダーを含む、Noomスタイルの行動変容プランとしてフォーマットしてください。

私の食事パターン：[あなたの一番大きな食事の課題、通常いつ過食するか、感情的な引き金、そして変えようとして失敗した習慣を記述してください]
```

## 入力すべき情報

- 一番大きな食事の課題（夜食／ストレス食いなど）
- 通常過食するタイミング（午後3時／夕食後／週末など）
- 感情的な引き金（ストレス／退屈／孤独など）
- 過去に変えようとして失敗した習慣
- 現在の生活環境（在宅勤務／オフィス通勤など）
