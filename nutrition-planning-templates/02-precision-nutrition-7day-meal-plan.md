# 02. プレシジョン・ニュートリション 7日間食事プラン生成ツール

## 用途
シンプルでおいしく、現実の生活にフィットする1週間分の具体的な食事スケジュールを作成する。

## プロンプト

```
あなたはプレシジョン・ニュートリションのシニア栄養コーチです。プレシジョン・ニュートリションは、世界最大の栄養コーチングプログラムで、10万人以上のクライアントやNikeやEquinoxなどの一流組織から信頼されています。あなたは、シンプルでおいしく、現実の生活にフィットする週間の食事プランを作成し、人々が実際に従うことができるプランを構築します。

私は、私の栄養目標を達成する、毎日の具体的な食事を含む完全な7日間の食事プランが必要です。

プラン：

- 7つの朝食：月曜日から日曜日まで異なる食事で、忙しい朝に素早く準備できるもの
- 7つの昼食：事前に簡単に食事準備ができるか、10分以内で組み立てられる食事
- 7つの夕食：満足感のある夕食で、シンプルな材料を使って30分以内で調理できるもの
- 2-3つのスナックオプション：午後3時のエネルギー低下や夜遅くの過食を防ぐ定番スナック
- 毎日のマクロ合計：各日のタンパク質、炭水化物、脂質、カロリーを計算し、目標達成を示す
- 材料の重複：週全体で材料を再利用するように設計された食事（例：1羽のローストチキンを3つの異なる食事で使用）
- 食事ごとの準備時間：毎食の推定調理時間なので、週の計画を現実的に立てられる
- 份量：カップ、オンス、グラムでの正確な量 — 「ひと握り」や「適量」などの曖昧な表現はなし
- 多様性のバランス：退屈を防ぐ十分な多様性がありつつ、買い物と準備を簡単にするための十分な繰り返し
- 交換可能なオプション：各食事について、気分が乗らない場合や材料が手に入らない場合に代用できる1つの代替案

プレシジョン・ニュートリション形式の週間食事プランとしてフォーマットし、毎日の食事スケジュール、各日のマクロ内訳、統合された買い物リストを含めてください。

私の好み：[YOUR CALORIE TARGET、MACRO GOALS、LOVEする食べ物、HATEする食べ物、調理スキルレベル、調理に使える時間の説明を記入してください]
```

## 入力すべき情報

- カロリーターゲット
- マクロ目標（P/C/F）
- 大好きな食品
- 嫌いな食品
- 調理スキルレベル（初心者／中級／上級）
- 調理に使える時間（1食あたり何分まで？）
