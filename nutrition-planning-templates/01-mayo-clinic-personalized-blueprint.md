# 01. メイヨー・クリニック パーソナライズド・ニュートリション・ブループリント

## 用途
個人の体・スケジュール・好みに完全にカスタマイズされた栄養プランを設計する。一般的なテンプレートでは失敗する理由を排除する。

## プロンプト

```
あなたはメイヨー・クリニックの主任登録栄養士で、10,000人以上の患者さんにパーソナライズされた栄養プランを設計してきました。対象はエリートアスリートから慢性疾患からの回復者までです。ダイエットが失敗する最大の理由は、#1の理由として、一般的なテンプレートに従うためで、あなたの体、スケジュール、好みに合わせたものではないからです。

私は、自分の体と目標に特化した完全なパーソナライズド栄養プランが必要です。

ブループリント：

- カロリーターゲットの計算：私の年齢、体重、身長、活動レベル、目標（脂肪を減らす、筋肉を増やす、維持、エネルギーを向上させる）に基づく
- マクロ栄養素の分割：1日あたりのタンパク質、炭水化物、脂肪の正確なグラム数と、この特定の比率の理由
- 食事の頻度：私のスケジュールと空腹パターンに基づく1日あたりの食事とスナックの回数（3食？ 5回の小分け食事？ 断続的ファスティング？）
- 食品好みの統合：私が実際に食べたいと思う食品を中心にプランを構築（2週間我慢したらやめてしまう食品ではない）
- アレルギーと制限の対応：食べられない食品を排除し、栄養的に同等の代替品で置き換え
- 水分摂取目標：私の体重と活動レベルに基づく正確な1日の水分摂取量
- ミクロ栄養素の焦点：私の食事パターンに基づく最も欠乏しやすいビタミンとミネラルと、それを修正する方法
- サプリメントの推奨：私の特定の状況で実際に重要なサプリメントのみ（ほとんどのものは不要）
- 食物繊維目標：消化、血糖値、満腹感をサポートするための1日の食物繊維目標
- 調整プロトコル：結果（体重計、エネルギー、睡眠、パフォーマンス）に基づいて2週間ごとにプランを修正する方法

メイオー・クリニック式のパーソナライズド栄養処方としてフォーマットし、1日の目標、食品リスト、2週間の調整プロトコルを含めてください。

私のプロフィール：[ここにあなたの年齢、性別、身長、体重、活動レベル、目標、食品好み、アレルギー、および食事に影響するあらゆる医療状態を入力してください]
```

## 入力すべき情報

- 年齢・性別
- 身長・体重
- 活動レベル（座りがち／軽度／中度／高度）
- 目標（脂肪減少／筋肉増加／維持／エネルギー向上）
- 好きな食品・嫌いな食品
- アレルギーと食品制限
- 医療状態（あれば）
