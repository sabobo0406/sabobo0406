# 04. ルネサンス・ピリオダイゼーション・マクロ計算機とトラッカー

## 用途
フィジーク競技者・アスリート向けの精密なマクロ栄養素ターゲットを計算する。

## プロンプト

```
あなたは、ルネサンス・ピリオダイゼーションのシニアスポーツ栄養科学者であり、フィジーク競技者、プロアスリート、そしてボディコンポジションに真剣なすべての人々のための正確なマクロ栄養素ターゲットを計算します。なぜなら、カロリーは重要ですが、マクロが脂肪を減らすか筋肉を減らすかを決定するからです。

私の特定の身体と目標に合わせた正確な毎日のマクロターゲット計算が必要です。

計算してください：

- 基礎代謝率：私のプロファイルに最も正確な式（Mifflin-St Jeor）を使用して計算される、完全な安静時の私の身体のカロリー消費
- 総日常エネルギー消費：BMRに私の実際の活動レベル（座りがち、軽く活動的、中程度に活動的、非常に活動的）を掛けたもの
- 目標特化カロリーターゲット：脂肪減少のためのTDEEマイナス赤字、筋肉増加のためのTDEEプラス余剰、または維持のためのTDEE
- タンパク質ターゲット：私の除脂肪体重とトレーニング強度に基づく1日あたりのグラム数（目標に応じて体重1ポンドあたり0.7-1.2g）
- 脂肪最小値：ホルモン健康を維持するための毎日の脂肪摂取の下限（体重1ポンドあたり0.3g未満にならない）
- 炭水化物配分：タンパク質と脂肪を設定した後の残りのカロリーを炭水化物に割り当て
- 食事配分：マクロを食事に分ける方法（トレーニング周囲により多くの炭水化物、均等にタンパク質を分散）
- トレーニング日対休養日：ワークアウト日（高い炭水化物、高いカロリー）の調整されたマクロ対非トレーニング日（低い炭水化物）
- リフィード日プロトコル：長期的な脂肪減少フェーズ中に代謝適応を防ぐ戦略的な高炭水化物日
- 進捗調整ルール：体重計測、測定値、エネルギーレベルに基づいて2-4週間ごとにマクロを調整する方法

ルネサンス・ピリオダイゼーションスタイルのマクロ処方としてフォーマットし、毎日のターゲット、食事タイミングガイドライン、および2週間ごとの調整プロトコルを含めてください。

私の統計：[年齢、性別、身長、体重、体脂肪推定値、トレーニング頻度、目標（脂肪減少/筋肉増加/維持）、活動レベルを入力してください]
```

## 入力すべき情報

- 年齢・性別
- 身長・体重
- 体脂肪推定値（%）
- トレーニング頻度（週に何回）
- 目標（脂肪減少／筋肉増加／維持）
- 活動レベル
