# dashboard — 売上・フォロワー数ダッシュボード

CSV に記録した売上やフォロワー数などの数字を、**一画面の HTML ダッシュボード**にまとめるツールです。
サーバー不要・ブラウザで開くだけで見られます。

## 使い方

1. `data/metrics.csv` に日々の数字を記録(列は自由に追加できます)

```csv
date,売上,Instagramフォロワー,YouTube登録者
2026-06-08,12000,1520,340
2026-06-09,8500,1535,342
2026-06-10,21000,1568,351
```

- 1列目は `date`(YYYY-MM-DD)固定
- 2列目以降は好きな指標を好きなだけ。列名がそのままダッシュボードの表示名になります

2. 生成

```bash
python build_dashboard.py
```

3. `dashboard.html` をブラウザで開く

## 表示内容

- 指標ごとのサマリーカード(最新値・前回比・直近7件の増減)
- 指標ごとの推移グラフ(Chart.js / CDN 読み込み)

## スプレッドシートと連携したい場合

Google スプレッドシートで管理している場合は、「ファイル → ダウンロード → CSV」で書き出して
`data/metrics.csv` に置き換えれば、そのまま使えます。
